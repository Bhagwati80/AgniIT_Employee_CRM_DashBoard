# AgniIT Employee Management Portal

A modern Employee Management and CRM Dashboard built with Next.js and TypeScript for managing employees, attendance, tasks, leave requests, and organizational workflows.

## Features

* Secure Login Authentication
* Employee Management
* Attendance Tracking
* Task Management
* Leave Management
* Reports Dashboard
* Settings Management
* Responsive User Interface

## Tech Stack

* Next.js
* React
* TypeScript
* Tailwind CSS
* Context API
* Local Storage

## Installation

```bash
git clone https://github.com/Bhagwati80/AgniIT_Employee_CRM_DashBoard.git
cd AgniIT_Employee_CRM_DashBoard
npm install
npm run dev
```

Open:

```txt
http://localhost:3000
```
